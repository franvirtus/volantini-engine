🤖 Guida — Come usare l'AI

Copia questi prompt e usali con qualsiasi AI (ChatGPT, Claude, Gemini...). Prima approva il design completo, poi chiedi le due cose separatamente.

STEP 1 — Chiedi l'immagine senza testi
Genera lo stesso volantino ma SENZA nessun testo. Solo immagini, icone, sfondi, decorazioni grafiche. Nessuna parola scritta, nessuna lettera. Dimensioni: 1080x1530 pixel.
STEP 2 — Chiedi il YAML overlay
{% raw %}Analizza il volantino originale (con i testi) e genera il YAML overlay per il motore VolantiniEngine. Canvas: 1080x1530 pixel. Il formato YAML deve essere ESATTAMENTE questo: output_name: "nome_output" theme: text: "#COLORE_TESTO_SCURO" white: "#FFFFFF" gold: "#COLORE_ORO" green: "#COLORE_VERDE" font: "Georgia" variables: output_name: "nome_output" template_img: "template.jpg" titolo: "Valore di default" sottotitolo: "Valore di default" # aggiungi una variabile per ogni testo modificabile canvas: width: 1080 height: 1530 layers: # Prima layer: sempre l'immagine base - id: base type: image src: "{{template_img}}" x: 0 y: 0 w: 1080 h: 1530 fit: fill # Poi un layer per ogni testo: - id: titolo type: text text: "{{titolo}}" x: [pixel da sinistra] y: [pixel dall'alto] w: [larghezza area] h: [altezza area] font: "{{theme.font}}" bold: true size: [dimensione pt] color: "{{theme.text}}" align: left REGOLE: - Ogni testo modificabile deve avere la sua variabile in variables: - Misura le coordinate in pixel sul canvas 1080x1530 - Usa {{theme.colore}} per i colori quando possibile - Il layer "base" con l'immagine deve essere SEMPRE il primo{% endraw %}
Poi: incolla il YAML nella tab "Incolla YAML", carica l'immagine template, clicca "Carica campi" e modifica i valori che vuoi cambiare.
